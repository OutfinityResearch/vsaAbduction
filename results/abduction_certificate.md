# Abduction certificate

This file is generated by `node scripts/run_experiment.js`. It is intended to make the claim inspectable rather than rhetorical.

## Pre-registered criterion

The run counts as a successful semantic abduction only if all of the following hold:

1. the abductor receives artifact descriptions and event labels, not the hidden oracle;
2. it introduces a latent semantic property as a sparse vector over words;
3. the property classifies held-out known-object events;
4. it extrapolates to larger counts;
5. it predicts events made only of novel objects described by words;
6. it proposes counterfactual repairs that the hidden oracle verifies;
7. the same method fails or generalizes poorly when labels are shuffled.
