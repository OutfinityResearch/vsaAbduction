# Manifest

This ZIP is intended to be unpacked directly as a Git repository.

## Main editable article

- `paper/eos_gate_semantic_abduction.docx` — editable English manuscript.
- `paper/article.md` — Markdown source-style copy of the paper text.

## Experiment code

- `scripts/run_experiment.js` — main runner.
- `scripts/generate_figures.js` — SVG figure generator.
- `test/verify_results.js` — verification report and assertions.
- `src/*.js` — experiment implementation. No npm dependencies.

## Specifications and methodology

- `docs/experiment_spec.md` — pre-specified hypotheses, success criteria, and failure modes.
- `docs/methodology.md` — experimental methodology.
- `docs/human_review_and_agent_use.md` — disclosure of coding-agent assistance and human-review expectations.
- `docs/reproducibility.md` — rerun checklist.

## Results from the included run

- `results/summary.json` — complete result object.
- `results/metrics.csv` — accuracy table.
- `results/hypothesis_comparison.csv` — MDL-style comparison.
- `results/abduction_certificate.md` — human-readable certificate.
- `results/verification_report.txt` — test output.
- `results/interventions.csv` — counterfactual repair examples.

## Figures

- `figures/*.svg` — source figures generated by Node.js.
- `figures/*.png` — rendered figures used by the DOCX manuscript.
